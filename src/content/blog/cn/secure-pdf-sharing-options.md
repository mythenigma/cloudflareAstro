# 安全分享 PDF 的几种方案

<div class="intro-panel">
  <p>MaiPDF 提供了多种安全选项，你可以单独使用，也可以组合搭配。<br/>这篇文章把每个选项拆开讲清楚，帮你快速决定"该开哪些"。</p>
</div>

## 全部安全选项一览

![安全设置界面](/麦瓜PDF/安全设置.png)

| 选项 | 功能 | 默认值 |
|------|------|--------|
| 打开次数 | 链接最多能被打开几次 | 不限 |
| 阅读时长 | 每次最多能看多久 | 不限 |
| 有效期 | 链接到期后自动失效 | 不限 |
| 保护模式 | 防下载/水印/截图干扰 | 不限制 |
| 邮箱验证 | 要求读者先验证邮箱 | 关闭 |
| 阅读提醒 | 有人打开时通知你 | 关闭 |
| 手机验证 | 要求输入手机号 | 关闭 |

## 逐项详解

### 打开次数

设成 5 次，第 6 次打开时显示"链接已失效"。最常用的安全选项。

- **场景**：报价单只给一个客户看，设 3-5 次
- **提示**：次数是总次数，不区分人

### 有效期

设定一个截止日期。过了这个日期，链接自动失效，无法再打开。

- **场景**：活动资料在活动结束后失效
- **提示**：可以和打开次数同时使用，哪个先到哪个生效

### 保护模式

四种级别可选：

| 级别 | 对方能做 | 对方不能做 |
|------|---------|------------|
| Unrestricted | 全部 | — |
| DynamicWatermark | 阅读 | 水印显示身份 |
| SecureView | 阅读 | 下载、打印 |
| FenceView | 局部阅读 | 截图也无用 |

![加密](/麦瓜PDF/宣传图/加密.png)

### 邮箱验证

开启后，对方必须输入邮箱地址，收到验证码后才能查看文件。你可以在后台看到具体是谁打开了。

### 阅读提醒

填写你的邮箱，每当有人打开链接时，你会收到一封通知邮件，包含打开时间和 IP 信息。

## 推荐组合

| 场景 | 推荐组合 |
|------|----------|
| 普通分享 | 打开次数 50 |
| 商务文件 | 次数 10 + 有效期 30 天 + SecureView |
| 机密文件 | 次数 3 + 有效期 7 天 + FenceView + 邮箱验证 |
| 付费内容 | 次数 1 + DynamicWatermark + 阅读提醒 |

不需要一次全开，选择适合你场景的组合就好。
