# 限制 PDF 查看次数：DRM 方案

<div class="intro-panel">
  <p>普通的"限次"是在线链接层面控制。DRM 更进一步——即使文件被下载到本地，也能限制打开次数。</p>
</div>

## 两种限次方式对比

| 方式 | 原理 | 需要联网 | 适用场景 |
|------|------|:--------:|----------|
| 在线限次 | 控制链接打开次数 | ✅ | 日常分享 |
| 离线 DRM 限次 | 加密 HTML 包内置次数限制 | ❌ | 内网/无网络环境 |

## 方式一：在线限次

这是最常用的方式。上传 PDF → 设置打开次数 → 生成链接。

![安全设置](/麦瓜PDF/安全设置.png)

对方通过链接在浏览器里看，每打开一次消耗一个配额。用完后链接失效。

**优点**：
- 操作简单，一分钟搞定
- 可随时调整次数或关闭链接
- 搭配有效期效果更好

**限制**：
- 对方必须联网才能看

## 方式二：离线 DRM 限次

把 PDF 打包成加密的 HTML 文件，发给对方后可以离线阅读。内置的 DRM 控制打开次数。

![离线方案](/麦瓜PDF/flowchart/zh-offline-version.svg)

**工作原理**：
1. 上传 PDF 到 MaiPDF DRM 页面
2. 选择"离线模式"
3. 设置打开次数
4. 下载加密 HTML 包
5. 把 HTML 文件发给对方
6. 对方用浏览器打开即可阅读

**优点**：
- 不需要联网
- 适合内网环境
- 文件加密，无法直接提取 PDF

**限制**：
- 次数用完后需要重新授权
- 需要浏览器支持 JavaScript

## 选哪个？

| 你的情况 | 选择 |
|---------|------|
| 对方能上网 | 在线限次 |
| 对方在内网/无网络 | 离线 DRM |
| 需要随时调整 | 在线限次 |
| 发一次就完事 | 离线 DRM |

大多数情况下，在线限次就够用。离线 DRM 适合特殊场景（如工厂、保密单位、考试系统）。
