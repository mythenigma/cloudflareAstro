# Free PDF Tools to Boost Your Productivity

You don't need expensive software for most PDF tasks. Between browser-based tools and free platforms, you can handle viewing, sharing, basic editing, and conversion without spending anything. Here's what works, what doesn't, and when you need to step up.

Most people only need 4-5 PDF capabilities regularly:

| Task | How Often | Free Solution Exists? |
|------|----------|----------------------|
| View PDFs | Daily | Yes — every browser does this |
| Share PDFs | Weekly | Yes — but with limitations |
| Convert to/from PDF | Occasionally | Yes — many free converters |
| Basic editing (annotate, sign) | Occasionally | Yes — free tools available |
| Secure sharing (access control) | When it matters | Partially — MaiPDF's free tier covers basics |

## Viewing: You Already Have It

Every modern browser (Chrome, Firefox, Edge, Safari) renders PDFs natively. No plugins, no installs.

For offline viewing or annotation:
- **SumatraPDF** (Windows) — fast, lightweight, no bloat
- **Preview** (macOS) — built-in, handles annotation
- **Evince / Okular** (Linux) — native PDF viewing

You probably don't need Adobe Acrobat Reader unless you're filling out complex forms with special fields.

## Conversion: Free Options That Work

### PDF to Word
- **Google Docs** — open a PDF, it converts to editable text (good for simple documents)
- **LibreOffice Draw** — opens and edits PDFs directly (free, cross-platform)

### Word/PowerPoint to PDF
- **Print to PDF** — built into Windows 10+, macOS, and Linux
- **LibreOffice** — File → Export as PDF from any document type

### Images to PDF
- **Windows Print to PDF** — select images, right-click, print to PDF
- **img2pdf** (Python) — lossless image-to-PDF conversion via command line

### PDF Compression
- **Ghostscript** (command line) — the standard for PDF optimization
- Browser-based compressors work but require uploading your document to someone else's server

## Annotation and Signing

For occasional annotations:
- **Browser-based signing** — most browsers let you fill and sign simple PDF forms
- **Xournal++** — free, open-source PDF annotation (handwriting, highlighting, notes)
- **macOS Preview** — sign with trackpad, add text, highlight

For heavy annotation work (legal review, academic markup), paid tools like PDF Expert or Adobe Acrobat become worth it.

- **Enterprise access control** — team-wide permission management
