### 方法三：技术实现方案

**JavaScript控制**：
```javascript
// 禁用打印功能
window.addEventListener('beforeprint', function(e) {
    e.preventDefault();
    alert('此文档禁止打印');
});

// 禁用打印快捷键
document.addEventListener('keydown', function(e) {
    if (e.ctrlKey && e.key === 'p') {
        e.preventDefault();
        alert('此文档禁止打印');
    }
});
```

**CSS控制**：
```css
/* 隐藏打印按钮 */
@media print {
    body {
        display: none !important;
    }
}

/* 打印时显示警告 */
@media print {
    .no-print {
        display: block !important;
        content: "此文档禁止打印";
    }
}
```
