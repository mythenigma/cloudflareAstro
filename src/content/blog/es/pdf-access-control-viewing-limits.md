## 🎯 ¿Qué son los Límites de Visualización?

Los límites de visualización le permiten establecer cuántas veces se puede acceder a un documento PDF compartido. Una vez alcanzado el límite, el enlace deja de funcionar.

## 🚀 Cómo Funciona

### Configuración de Límites

1. **Subir PDF**: Suba su documento PDF
2. **Configurar Límite**: Establezca el número máximo de visualizaciones
3. **Generar Enlace**: Cree un enlace compartible
4. **Monitorear**: Rastree el uso en tiempo real

### Ejemplos de Uso

- **Límite de 1**: Para documentos de un solo uso
- **Límite de 5**: Para documentos con acceso limitado
- **Límite de 10**: Para documentos con acceso moderado
