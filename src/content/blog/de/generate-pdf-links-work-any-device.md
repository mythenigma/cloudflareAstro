## 🚀 So Funktioniert Es

### Generierung

1. **PDF Hochladen**: Laden Sie Ihr PDF hoch
2. **Link Generieren**: Erstellen Sie einen universellen Link
3. **Teilen**: Teilen Sie den Link mit Empfängern
4. **Zugriff**: Empfänger können von jedem Gerät zugreifen

### Kompatibilität

- **Desktop**: Windows, Mac, Linux
- **Mobile**: iOS, Android
- **Tablet**: iPad, Android Tablets
- **Browser**: Chrome, Safari, Firefox, Edge
