# Maiimg 이벤트 사진 배포 가이드

<div class="intro-panel">
  <p>행사 당일 참가자가 사진을 바로 볼 수 있도록 하면서, 공개 범위를 통제하는 것은 쉽지 않습니다. <strong>Maiimg</strong>는 QR 코드 기반 갤러리로 속도와 보안을 동시에 확보합니다.</p>
</div>

![Maiimg Event Flow](/maiimg/maiimg-head.png)

## 갤러리 설계

| 대상 | 권장 설정 |
|------|-----------|
| 일반 참가자 | 조회 150회 / 30일 / 다운로드 OFF |
| VIP·스폰서 | 조회 50회 / 14일 / 다운로드 ON |
| 언론/PR | 조회 40회 / 7일 / 비밀번호 + 다운로드 ON |
| 내부 스태프 | 무제한 / 무기한 / 다운로드 ON |

## 운영 단계

1. **사진 선별** – 세션/시간대별로 10~20장 선정
2. **Maiimg 업로드** – 용도별로 갤러리 분리 (Day1, VIP 등)
3. **접근 제어** – 조회수, 만료일, 다운로드 여부 설정
4. **QR 코드 인쇄** – 리셉션, 포토월, 테이블 등 주요 동선에 설치
5. **메시지 배포** – SMS/메일/메신저로 링크 공유
6. **Analytics 확인** – 뷰 수, 인기 사진, 시간대 분석

## 현장 활용 아이디어

```
[포토월 옆 스탠드]
Scan & Save
이벤트 사진은 여기에서!
QR 코드
30일 후 자동 삭제
```

- 세션 사이 스크린에 QR 노출
- 네임택 뒷면에 QR 스티커 부착
- 스폰서 부스에서 전용 QR 카드 제공

## 메시지 템플릿

```
안녕하세요, Summit 2025 사진이 업데이트되었습니다.
https://maiimg.com/... (30일 후 자동 만료)
영업비밀 보호를 위해 외부 공유는 자제 부탁드립니다.
```

## Analytics로 얻는 인사이트

- 가장 많이 본 사진 → 홍보용 대표 컷 선정
- 피크 시간 → 팔로업 메시지 타이밍 조정
- 지역/디바이스 데이터 → 온라인 참가자 파악
- 스폰서 갤러리 조회수 → 스폰서 리포트 자료로 활용

## 문제 해결

| 문제 | 해결 방법 |
|------|-----------|
| QR 인식 불가 | 고해상도로 재생성, 대비 강화 |
| 용량 초과 | 갤러리를 일/주제별로 분할 |
| 무단배포 우려 | 조회수 제한 + 패스워드 적용 |
| 인터넷 불안정 | 사전 안내 + Wi-Fi/5G 가이드 제공 |

## 결론

Maiimg는 이벤트 사진 배포를 “즉시·안전·데이터 기반” 프로세스로 바꿔줍니다. 표준화된 갤러리와 QR 셋업을 준비해 두면 어느 행사에서도 동일한 품질로 서비스를 제공할 수 있습니다.
