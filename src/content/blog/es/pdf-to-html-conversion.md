## 🚀 Cómo Funciona

### Proceso de Conversión

1. **PDF Subir**: Suba su documento PDF
2. **Conversión**: Convierta a HTML automáticamente
3. **Obtener HTML**: Reciba página HTML
4. **Compartir**: Comparta la página HTML

### Características HTML

- **Responsive Design**: Adaptación automática
