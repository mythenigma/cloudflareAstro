# Vérification par e-mail pour l’accès à un PDF (flux rapide)

<div class="intro-panel">
  <p>La vérification e-mail est une manière “légère” de réduire l’accès via transfert de lien, sans rendre le partage compliqué.</p>
</div>

![Workflow de vérification e-mail](/diagram/en/email-verification-access-control-workflow.svg)

## Le flux (rapide)

- Le lecteur saisit son e-mail
- Reçoit un code à usage unique
- Vérifie, puis ouvre le PDF

## Idéal pour

- Propositions, devis, livrables client
- Documents RH et politiques internes
