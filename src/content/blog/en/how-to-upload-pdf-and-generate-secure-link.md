1. Toggle "Require email verification" ON

3. Restrict to specific domains (optional)
4. Enable email-based access control

**Password Protection:**
1. Enable "Password protection"
2. Set a strong password
3. Share password separately

**Phone Verification:**

### View and Access Limits

**Total View Limits:**
1. Enable "Limit total views"
2. Set maximum number (e.g., 10, 20, 50)
3. Control document distribution
4. Prevent unlimited access

**Per-User View Limits:**
1. Enable "Limit views per user"
2. Set per-user limit (e.g., 3, 5)
3. Prevent single user exhaustion
4. Better access management

**Expiration Date:**
1. Enable "Set expiration date"
2. Choose expiration date and time
3. Automatic link deactivation
4. Time-limited access

### Download and Print Prevention

**Download Prevention:**
1. Toggle "Prevent downloads" ON
2. Remove download button
3. Block right-click save
4. Prevent file saving

**Print Prevention:**

## Best Practices

### 1. Always Set Access Control

**Essential:**
- ✅ Enable email verification
- ✅ Set password for sensitive docs
- ✅ Use domain restrictions for teams
- ✅ Enable phone verification for confidential files

### 2. Set Appropriate Limits

**For Confidential Documents:**
- 1-3 views per user
- 5-10 total views
- 7-day expiration

**For Business Documents:**
- 3-5 views per user
- 10-20 total views
- 14-30 day expiration

**For Training Materials:**
- 5-10 views per user
- 20-50 total views
- 30-90 day expiration

### 4. Use Watermarks
