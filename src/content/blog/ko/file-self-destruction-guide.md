## 파일 자동 삭제란 무엇인가?

**파일 자동 삭제**는 특정 시간 후 파일을 자동으로 삭제하거나 비활성화하는 보안 기능입니다. 사용자가 기밀 문서를 안전하게 공유하면서 파일 만료를 완전히 제어할 수 있도록 합니다.

**주요 기능:**
- ✅ **자동 삭제** - 특정 시간 후 파일 자동 삭제
- ✅ **만료 제어** - 파일 만료를 정확하게 제어
- ✅ **문서 보호** - 기밀 문서 보호
- ✅ **추적 가능성** - 파일 만료 모니터링
