# 设置查看限制防止未授权访问

<div class="intro-panel">
  <p>链接泄露了怎么办？设好<strong>查看限制</strong>，即使链接被转发，超过次数也打不开。</p>
</div>

## 三道防线

### 第一道：限制打开次数

![安全设置](/麦瓜PDF/安全设置.png)

在"可打开次数"栏填入数字。比如：

| 场景 | 建议次数 | 理由 |
|------|---------|------|
| 一对一发给客户 | 3-5 次 | 对方看几次就够了 |
| 小团队内部 | 10-20 次 | 覆盖团队人数 |
| 公开分发 | 50-100 次 | 满足传播需要 |
| 付费阅读 | 1 次 | 看完即消 |

超出次数后，任何人打开链接都会看到"已失效"。

### 第二道：设置有效期

设定一个截止时间。过了这个时间，即使还有剩余次数，链接也自动失效。

常见场景：
- 活动资料：活动结束后失效
- 合同草稿：签约前 7 天有效
- 临时文件：24 小时后自动销毁

![阅后即焚](/麦瓜PDF/宣传图/阅后即焚.png)

### 第三道：身份验证

开启邮箱验证后，对方必须先输入邮箱 → 收到验证码 → 输入验证码，才能查看文件。

这样做的好处：
1. 你明确知道**谁**看了你的文件
2. 陌生人即使拿到链接也打不开
3. 后台有完整的访问记录

## 操作步骤

1. 上传 PDF 到 [MaiPDF](https://maipdf.com)
2. 在安全设置页配置以上三项
3. 生成链接发送

![生成结果](/麦瓜PDF/生成结果.png)

## 分享后监控

登录后台查看每个链接的状态：

![文件列表](/麦瓜PDF/用户下的上传列表.png)

- 已用次数 / 总次数
- 最后访问时间
- 是否过期

发现异常（比如次数消耗异常快），可以立即关闭链接。

## 总结

查看限制 = 次数 + 有效期 + 身份验证。三道防线叠加，即使链接泄露，未授权的人也进不来。
