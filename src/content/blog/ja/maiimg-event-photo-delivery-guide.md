# Maiimgイベント写真配布ガイド：QR×アクセス制御で即日共有

<div class="intro-panel">
  <p>イベント参加者は「その日のうちに写真を見たい」。一方で主催者とフォトグラファーは、公開範囲やブランド統制を守りたい。<strong>Maiimg</strong>ならギャラリー単位で閲覧制限を掛けつつQRコードでスピード配布が可能です。</p>
</div>

![Maiimg Event Gallery](/maiimg/maiimg-head.png)

## シナリオ別ギャラリー

| イベント | 目的 | 推奨設定 |
|----------|------|----------|
| ビジネスカンファレンス | セッションハイライト共有 | 閲覧150回 / 30日 / DL× |
| 展示会・EXPO | 出展社向け素材提供 | 閲覧120回 / 45日 / DL◯ |
| ウェディング | 新郎新婦・ゲストへの速報 | 閲覧80回 / 14日 / DL×、パスワード付 |
| ブランドポップアップ | SNS投稿用 | 閲覧200回 / 7日 / DL◯ / QR手配 |

## ワークフロー

1. **1日目ハイライトを選定**（10～20枚）
2. **Maiimgに一括アップロード**（最大25枚）
3. **閲覧回数・有効期限・DL可否を設定**
4. **QRコードを出力し、会場導線に設置**
5. **メール/SNSでもリンク配布**
6. **アクセスログで効果測定**

## QRコード活用例

- 受付・フォトスポットに「写真はこちら」の立て看板
- 参加証やノベルティにステッカー貼付
- ステージ転換中にスクリーン表示
- フォローアップメールにQR画像を添付

## テンプレート（終了後メール）

```
件名：◯◯カンファレンス フォトギャラリー

ご参加ありがとうございました。イベント写真は以下からご覧いただけます。
https://maiimg.com/...（30日でアクセス停止）
※SNSへ投稿される場合は #Event2025 をご利用ください。
```

## 解析で見るべき指標

- 総ビュー数＆ユニークユーザー
- ピーク時間（例：懇親会後）
- よく閲覧された写真（次回の撮影ポイント）
- 地域データ（オンライン参加者の把握）

## トラブルシューティング

| 問題 | 対処 |
|------|------|
| QRが読み取れない | 高解像度で再出力、背景とのコントラスト調整 |
| ギャラリー上限 | 日別/セッション別に複数ギャラリーを作成 |
| 無断転送が不安 | 閲覧回数を少なめに設定 + パスワードを共有 |
| ネット回線が不安定 | 事前に「Wi-Fi/4Gの推奨」をアナウンス |

## まとめ

Maiimgを使えば、イベント写真を「即日」「安全」「可視化」付きで届けられます。ギャラリー設定とQR配布を標準化し、イベント価値を最大化しましょう。
