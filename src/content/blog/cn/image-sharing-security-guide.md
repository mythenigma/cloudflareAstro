# 图片分享安全指南

<div class="intro-panel">
  <p>图片一旦发出去就失控了？不一定。用对工具，你可以控制谁能看、看几次、能不能保存。</p>

## 图片分享的安全风险

| 风险 | 说明 |
|------|------|
| 随意转发 | 你发给一个人，他转给了所有人 |
| 截屏保存 | 对方截图后你完全不知道 |
| 盗用 | 你的作品被别人拿去用 |
| 永久留存 | 你想删除时已经传开了 |

## MaiImg 的安全分享方案

MaiImg 是专门为图片设计的安全分享工具，和 MaiPDF 同属一个产品线。

![MaiImg 首页](/maiimg/static-images/cn/麦瓜二维码首页.png)

### 核心安全功能

| 功能 | 作用 |
|------|------|
| 打开次数限制 | 超过次数后图片不可访问 |
| 有效期 | 过期自动失效 |
| 一键删除 | 随时可以关闭分享 |
| 访问追踪 | 知道谁在什么时候看了 |

## 不同场景的安全建议

| 场景 | 推荐设置 |
|------|---------|
| 发产品图给客户 | 打开 30 次，90 天有效 |
| 发设计稿给同事 | 打开 10 次，30 天有效 |
| 发私密照片给朋友 | 打开 3 次，7 天有效 |
| 发证件照给机构 | 打开 5 次，14 天有效，用完删除 |

## 如果需要删除

不想让别人继续看？两种方式：

1. 在管理页面一键删除
2. 设置有效期，自动失效

![删除](/maiimg/static-images/cn/一键删除图片.png)

## 总结

图片分享不等于失控。设好打开次数和有效期，看完就失效，用完就删除。
