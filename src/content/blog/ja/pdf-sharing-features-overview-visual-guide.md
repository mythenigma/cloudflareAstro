# PDF共有機能の全体像（ビジュアルガイド）

<div class="intro-panel">
  <p>長い解説より、全体像を一気に掴みたい人向け。共有に必要な主要機能を一枚にまとめた図です。</p>
</div>

![PDF sharing features overview](/diagram/en/features-overview.svg)

## 図が示すポイント

- **オンライン制御**：閲覧回数、期限、本人確認、トラッキング
- **オフライン配布**：ネットに依存しない自己完結パッケージ
- **セキュリティレイヤー**：共有を簡単にしつつリスクを下げる仕組み

## 迷ったらこの基準

- **オンライン**：リアルタイム制御と分析が必要
- **オフライン**：ネットに頼れず、単一ファイルで配布したい
