## 🎯 Was sind Ansichtslimits?

Ansichtslimits ermöglichen es Ihnen festzulegen, wie oft auf ein geteiltes PDF-Dokument zugegriffen werden kann. Sobald das Limit erreicht ist, funktioniert der Link nicht mehr.

## 🚀 So Funktioniert Es

### Limit-Konfiguration

1. **PDF Hochladen**: Laden Sie Ihr PDF hoch
2. **Limit Festlegen**: Legen Sie die maximale Anzahl der Ansichten fest
3. **Link Generieren**: Erstellen Sie einen teilbaren Link
4. **Überwachen**: Verfolgen Sie die Nutzung in Echtzeit

### Nutzungsbeispiele

- **Limit von 1**: Für Einmal-Dokumente
- **Limit von 5**: Für Dokumente mit begrenztem Zugriff
- **Limit von 10**: Für Dokumente mit moderatem Zugriff

## 💡 Anwendungsfälle

### Vertrauliche Dokumente

- Vertrauliche Berichte

### Temporäre Materialien
