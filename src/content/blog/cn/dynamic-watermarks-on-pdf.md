# PDF 动态水印的用法与价值

<div class="intro-panel">
  <p>静态水印人人会加，但动态水印不一样——它根据查看者的身份实时生成，每个人看到的水印内容不同。</p>
</div>

## 动态水印 vs 静态水印

| 特性 | 静态水印 | 动态水印 |
|------|---------|---------|
| 内容 | 固定文字/LOGO | 查看者邮箱/IP/时间 |
| 可追溯 | ❌ 无法定位泄露者 | ✅ 精确到个人 |
| 去除难度 | 中 — 有工具可以处理 | 高 — 每次渲染不同 |
| 适用场景 | 品牌标识 | 机密文档防泄密 |

## 动态水印能显示什么？

MaiPDF 的动态水印支持以下变量：

- **查看者邮箱**：如果开启了邮箱验证
- **查看者 IP 地址**
- **查看时间**：精确到分钟
- **自定义文字**：公司名、文件密级等

![安全设置](/麦瓜PDF/安全设置.png)

## 典型使用场景

### 投资材料

投资人收到的 BP 上会显示各自的邮箱，一旦泄露可以追溯到具体是谁。

### 内部培训文件

员工查看培训资料时，水印显示工号或邮箱，防止截图外传。

### 合同审批

每位审批人看到的合同都带有自己的标识，确保审计可追溯。

## 开启步骤

1. 上传 PDF 到 MaiPDF
2. 在安全设置中开启"动态水印"
3. 选择水印内容（邮箱/IP/时间/自定义文字）
4. 生成链接

![分享控制](/麦瓜PDF/宣传图/分享之后能控制.png)

## 配合其他安全功能

动态水印通常和以下功能一起使用效果更好：

| 功能组合 | 效果 |
|---------|------|
| 动态水印 + 邮箱验证 | 水印显示真实邮箱，100% 可追溯 |
| 动态水印 + SecureView | 防截屏 + 水印双重保障 |
| 动态水印 + 打开次数限制 | 限量查看 + 可追溯 |
| 动态水印 + 有效期 | 过期自动失效 + 在有效期内可追溯 |

![加密保护](/麦瓜PDF/宣传图/加密.png)

## 总结

动态水印是防泄密的最后一道防线。即使有人截图分享，水印上的个人信息也能帮你定位到泄露源头。
