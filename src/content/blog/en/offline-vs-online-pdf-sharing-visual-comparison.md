# Offline vs Online PDF Sharing (Visual Comparison)

<div class="intro-panel">
  <p>Online is best for real-time control and tracking. Offline is best for distributing a self-contained file in restricted environments.</p>
</div>

![Offline vs online PDF comparison](/diagram/en/offline-vs-online-pdf-comparison.svg)

## Choose online when you need

- Verification, view limits, expiration
- Logs and analytics
- Instant changes (disable/update anytime)

## Choose offline when you need

- A single package to distribute
- No reliance on internet access
- Controlled environments (training rooms, restricted devices)
