## 🛠️ 三步搞定：从PDF到二维码

### 第一步：上传你的PDF

**超级简单的上传过程**：
1. 打开MaiPDF网站
2. 把PDF文件拖进去（或者点击上传）
3. 等几秒钟处理完成

![上传界面](/cn2025May/maipdf_upload_pdf_cn.png)

**小贴士**：
- 📄 支持各种PDF文件，没有页数限制
- 💪 最大支持200MB（一般的文档都够用）
- 🚀 上传速度很快，几秒钟就完成

### 第二步：设置访问权限

上传完成后，你可以设置谁能看这个文档：

<div class="settings-container">

<div class="setting-card">
**🚪 访问方式**
- **无需验证**：任何人扫码就能看（适合公开资料）
- **手机验证**：输入手机号收验证码才能看（更安全）
- **邮箱验证**：输入邮箱收确认邮件（适合收集联系方式）
</div>

<div class="setting-card">
**⏰ 时间限制**
- **永久有效**：什么时候扫都能看
- **设置到期日期（仅离线版）**：比如一周后自动失效
- **限制次数**：每个人最多看3次
</div>

</div>

![设置界面](/cn2025May/upload_good_maipdf_cn.png)

### 第三步：生成二维码

设置完成后，系统会给你：
- 🔗 一个短链接（备用）
- 📱 一个二维码图片（主要用这个）
- 📊 一个管理页面（查看谁看了）

## 🔧 实用技巧和注意事项

### 📐 二维码尺寸怎么选？

<div class="size-guide">

**手机分享**：200×200像素就够了
- 微信聊天发送的最佳尺寸
- 扫描距离：10-30厘米

**投影展示**：400×400像素
- 会议室投影的标准尺寸
- 扫描距离：2-5米

**打印使用**：600×600像素
- A4纸张打印不会糊
- 扫描距离：30-50厘米

**海报宣传**：800×800像素以上
- 展会、活动现场使用
- 扫描距离：1-3米

</div>

### 🎯 提高扫码成功率的小窍门

**环境方面**：
- ☀️ 避免强光反射，找个光线适中的地方
- 🎨 确保二维码和背景有足够对比度
- 📱 保持手机稳定，不要抖动

**设计方面**：
- 🎯 在二维码旁边加上"扫码查看"的提示
- 📍 告诉大家扫码后能获得什么
- 💡 如果是现场使用，可以口头提醒一下

### 📊 如何知道有多少人看了？

MaiPDF会自动记录访问数据：
- 📈 总共有多少人扫码
- ⏰ 什么时候扫的码
- 📱 用的什么设备（iPhone还是安卓）
- 🌍 大概在哪个城市

![扫码统计](/cn2025May/acess_result.png)

这些数据可以帮你了解：
- 你的文档是否受欢迎
- 最佳的分享时间
- 哪些内容更吸引人

## 🔄 文档更新和管理

### 📝 内容需要修改怎么办？

**好消息**：二维码不用重新生成！

1. 登录MaiPDF管理后台
2. 找到你的文档
3. 点击"替换文件"
4. 上传新版本的PDF
5. 完成！二维码还是原来那个，但内容已经更新了

![替换文件](/cn2025May/changefilesetting.png)

### ⏰ 临时延长有效期（仅离线版）

如果文档快过期了，但还需要让人继续看：
- 进入管理后台
- 修改有效期（仅离线版）设置
- 立即生效，不需要重新生成二维码

### 🔒 临时修改访问权限

比如原本设置无需验证，现在想加上手机验证：
- 在设置页面修改验证方式
- 之前扫过码的人不受影响
- 新的访问者需要按新规则验证

## ⚠️ 安全使用建议

### 📋 不同类型文档的建议设置

**公开资料**（如产品介绍、活动宣传）：
```
验证方式：无需验证
有效期（仅离线版）：根据需要，可以设置永久
访问次数：不限制
```

**内部资料**（如会议纪要、内部通知）：
```
验证方式：邮箱验证
有效期（仅离线版）：1-2周
访问次数：每人3-5次
```

**私人文档**（如简历、个人作品）：
```
验证方式：手机验证
有效期（仅离线版）：根据需要设置
访问次数：限制次数，避免过度传播
```

### 🛡️ 防止文档泄露的小技巧

**添加水印**：
- 可以显示查看者的手机号或邮箱
- 即使截图也能追踪到是谁泄露的

**限制下载**：
- 只允许在线查看，不允许下载
- 减少文档被大量复制的风险

**定期检查**：
- 查看打开统计，发现异常及时处理
- 对于重要文档，建议设置较短的有效期（仅离线版）

## 💡 常见问题解答

**Q：生成二维码需要付费吗？**
A：基础功能免费，高级功能可能需要付费。一般日常使用免费版就够了。

**Q：扫码的人需要安装特殊APP吗？**
A：不需要，用微信扫一扫或者手机自带的扫码功能就行。

**Q：文档会保存在哪里？**
A：文档保存在MaiPDF的服务器上，有安全保障。

**Q：可以看到谁扫了码吗？**
A：可以看到扫码的统计数据，但是个人隐私信息是保护的。

**Q：二维码会过期吗？**
A：看你怎么设置，可以永久有效，也可以设置到期日期（仅离线版）。

**Q：一个PDF可以生成多个二维码吗？**
A：可以的，同一个文档可以生成不同设置的二维码。
