## 🚀 So Funktioniert Es

### Bildspeicherung

1. **Bilder Hochladen**: Laden Sie Ihre Bilder hoch
2. **Organisieren**: Organisieren Sie Bilder in Galerien
3. **Teilen**: Generieren Sie Links zum Teilen
4. **Verwalten**: Verwalten Sie Ihre Bildsammlung

### Funktionen

- **Kostenlose Speicherung**: Unbegrenzte Speicherung
- **Galerien**: Organisieren Sie Bilder in Galerien
- **Links**: Generieren Sie teilbare Links
- **QR-Codes**: Erhalten Sie QR-Codes für Bilder
