# MaiImg 二维码生成

<div class="intro-panel">
  <p>二维码是线下分享图片最方便的方式。MaiImg 上传图片后自动生成二维码，扫一扫就能看到高清原图。</p>
</div>

## 为什么用二维码分享图片？

| 场景 | 传统方式 | 二维码方式 |
|------|---------|-----------|
| 展会产品图 | 打印一堆资料 | 一个二维码展板 |
| 餐厅菜单 | 每桌放纸质菜单 | 桌上贴二维码 |
| 课堂教学 | 投屏或发群 | 学生扫码看大图 |
| 活动现场 | 现场传文件 | 扫码即看 |
| 名片 | 没法放作品 | 扫码看作品集 |

## 生成步骤

### 1. 上传图片

打开 MaiImg，选择或拖入图片。

![首页](/maiimg/static-images/cn/麦瓜二维码首页.png)

### 2. 配置选项

可以设置打开次数和有效期（也可以不设，用默认值）。

![设置](/maiimg/static-images/cn/设置打开次数.png)

### 3. 获取二维码

系统自动生成链接和二维码。右键保存二维码图片即可使用。

![结果](/maiimg/static-images/cn/生成结果.png)

## 二维码的使用技巧

- **打印前测试**：先用手机扫一下确认正常
- **保留管理权**：注册账号后可以随时删除或修改
- **控制大小**：印刷时二维码建议不小于 3cm × 3cm
- **加文字说明**：二维码旁边写上"扫码查看图片"

## 二维码的安全管理

分享出去的二维码也能管控：

| 操作 | 说明 |
|------|------|
| 限制打开次数 | 活动结束后自动失效 |
| 设有效期 | 展会期间有效，展后过期 |
| 查看统计 | 多少人扫了、什么时候扫的 |
| 一键删除 | 不想让人看了就删掉 |

![追踪统计](/maiimg/result_tracking.png)

## 总结

上传图片 → 自动生成二维码 → 印出来或发出去。线下分享图片，二维码是最简单的方式。
