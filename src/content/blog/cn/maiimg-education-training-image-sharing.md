# Maiimg教育培训课程图片分享指南

<div class="intro-panel">
  <p>培训机构每天都有课程回顾、学员作品、活动花絮要分享，但又要保护学员隐私。利用<strong>Maiimg</strong>可以按班级或课程创建图库，设定查看次数、到期时间、是否允许下载，并用链接或二维码发送给学员与家长。</p>
</div>

## 场景拆分

| 场景 | 内容 | 权限建议 |
|------|------|----------|
| K12培训 | 课堂照片、作业示例 | 查看30次、下载关闭、7天到期 |
| 职业培训 | 项目作品、证书合影 | 查看80次、下载允许、30天到期 |
| 校园活动 | 开幕式、比赛花絮 | 查看150次、下载关闭、14天到期 |
| 线上课程 | PPT图、社群素材 | 查看300次、下载允许、周期内有效 |

## 操作流程

1. **按班级/课程创建图库**，例如 `2025春季-UI设计班-Week2`
2. **上传图片**（每次最多25张，推荐精选）
3. **配置访问规则**：
   - 查看次数限制，防止链接外传
   - 到期时间，与课程周次同步
   - 下载是否开启，根据隐私需求决定
   - 必要时启用密码
4. **推送方式**：
   - 微信/企业微信/钉钉班级群中发送链接
   - 家长会上使用二维码贴纸
   - 在线课程平台的公告栏插入链接
5. **查看数据**：掌握是否有学员未阅读，便于提醒

## 隐私与安全

- 上传前可在图片上打水印或马赛克
- 使用短期到期（如7天）防止长期扩散
- 链接中添加提示：“仅限本班学员查看，请勿外传”
- 活动结束后删除图库，符合隐私合规

## 二维码在课堂的玩法

```
▼ 今日课程素材
扫码即可下载练习图
有效期：7天
```

- 贴在教室白板或讲义上
- 结课时给学员一张“复习卡”带走
- 夏令营/冬令营用二维码汇集每日照片

## 数据驱动运营

- 通过查看次数判断学员复习积极度
- 家长是否打开链接 → 做好回访
- 对最受欢迎的作品进行二次内容营销

## 常见问题

| 问题 | 解决 |
|------|------|
| 学员说链接无效 | 查看次数或到期 | 手动增加或延长，再次发送 |
| 图片太多 | 25张限制 | 按“课堂回顾”“优秀作品”拆分多个图库 |
| 担心泄露 | 设置密码 + 短期访问 + 关闭下载 |
| 网速慢 | 先压缩或使用WebP格式，提高加载速度 |

## 结语

Maiimg让教育培训机构的图片分享更加：

- **规范**：统一命名、统一权限
- **高效**：批量上传、自动生成链接/二维码
- **可控**：随时删除、查看数据、保护隐私

无论是短期课程还是长期班级，都可以用Maiimg打造专业的图片交付体验。
