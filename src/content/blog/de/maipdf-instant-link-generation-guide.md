## 🚀 So Funktioniert Es

### Einfacher Prozess

1. **PDF Hochladen**: Laden Sie Ihr PDF hoch
2. **Link Generieren**: Klicken Sie auf "Link Generieren"
3. **QR-Code Erhalten**: Erhalten Sie automatisch einen QR-Code
4. **Teilen**: Teilen Sie Link oder QR-Code

### Optionen

- **Direkter Link**: Teilen Sie den Link direkt
- **QR-Code**: Verwenden Sie QR-Code zum Teilen
- **Beides**: Verwenden Sie Link und QR-Code
