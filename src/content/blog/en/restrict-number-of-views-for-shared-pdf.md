# Restrict Number of Views for a Shared PDF

View limits are a direct way to prevent indefinite access after sharing.

## Recommended use

- Confidential reports
- Limited review rounds
- Paid or restricted content

## Setup workflow

1. Upload PDF.
2. Set view limit.
3. Add expiry and optional download restriction.
4. Share segmented links.
5. Rotate link when scope changes.

## Final takeaway

View-limit policy is simple and effective when applied consistently.
