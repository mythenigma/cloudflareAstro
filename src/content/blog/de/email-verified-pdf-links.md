# E-Mail-verifizierte PDF-Links: Zugriff bei Weiterleitung stoppen

<div class="intro-panel">
  <p>Wenn ein PDF-Link weitergeleitet wird, verlierst du Kontrolle. E-Mail-Verifizierung bindet Zugriff an eine Person, nicht an eine URL.</p>
</div>

![Email Verification Access Control Workflow](/diagram/en/email-verification-access-control-workflow.svg)

## Wie es funktioniert (einfach)

- Link teilen
- Viewer gibt E-Mail ein
- Code wird gesendet
- Nur verifizierte Viewer können öffnen

## Wann es sinnvoll ist

- Proposals und Angebote
- HR-Dokumente
- Kunden-Deliverables
