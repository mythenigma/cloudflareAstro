## Recipient Instructions

### Essential Instructions

**Opening Instructions:**

1. **Extract ZIP File**
   ```
   - Locate downloaded ZIP file
   - Right-click and select "Extract All"
   - Choose extraction location
   - Wait for extraction to complete
   ```

2. **Open HTML File**
   ```
   - Navigate to extracted folder
   - Find "index.html" file
   - Double-click to open
   - File opens in default browser
   ```

3. **Network Verification**
   ```
   - Initial network connection required
   - Verification process completes
   - After verification, can view offline
   - Logs sync when device is online
   ```

4. **View PDF**
   ```
   - PDF displays in browser
   - Navigate using controls
   - View restrictions apply
   - Can view offline after verification
   ```
