## 🚀 So Funktioniert Es

### Prozess

1. **Bilder Auswählen**: Wählen Sie mehrere Bilder aus
2. **Galerie Erstellen**: Erstellen Sie eine Galerie
3. **Link Generieren**: Generieren Sie einen Galerie-Link
4. **Teilen**: Teilen Sie die Galerie mit einem Klick

### Galerie-Features

- **Mehrere Bilder**: Fügen Sie so viele Bilder hinzu wie Sie möchten
- **Organisation**: Organisieren Sie Bilder in Galerien
- **Anpassung**: Passen Sie Galerie-Darstellung an
- **Sharing**: Teilen Sie mit einem Link
