# MaiImg 安全图片分享与二维码功能介绍

<div class="intro-panel">
  <p>MaiImg 是一个图片安全分享工具。上传图片后自动生成链接和二维码，接收者扫码或点链接就能查看。</p>
</div>

## MaiImg 是什么？

MaiImg 是 MaiPDF 的姊妹产品，专门解决图片分享的痛点：

- 发微信会被压缩
- 发邮件附件太大
- 发网盘对方要登录
- 发出去后无法管控

![MaiImg](/maiimg/static-images/cn/麦瓜二维码首页.png)

## 核心功能

| 功能 | 说明 |
|------|------|
| 图片上传 | 支持常见图片格式 |
| 自动生成链接 | 上传即得分享链接 |
| 自动生成二维码 | 适合线下场景 |
| 打开次数限制 | 可设定最大查看次数 |
| 有效期 | 过期自动失效 |
| 访问追踪 | 查看者记录可查 |
| 一键删除 | 随时终止分享 |

## 二维码分享的优势

二维码特别适合这些场景：

- **线下活动**：展板上放二维码，扫码看高清大图
- **名片**：扫码查看作品集
- **产品包装**：扫码查看使用说明图
- **课堂教学**：老师分享教学图片

![扫码分享](/maiimg/static-images/cn/宣传/扫码分享.jpg)

## 使用流程

1. 打开 MaiImg 网站
2. 上传图片
3. 设置安全选项（可选）

![设置](/maiimg/static-images/cn/设置打开次数.png)

4. 获取链接和二维码

![结果](/maiimg/static-images/cn/生成结果.png)

5. 分享给需要的人

## 与普通图床的区别

| 对比 | 普通图床 | MaiImg |
|------|---------|--------|
| 打开次数限制 | ❌ | ✅ |
| 有效期 | ❌ | ✅ |
| 访问追踪 | ❌ | ✅ |
| 一键删除 | ⚠️ 部分支持 | ✅ |
| 二维码 | ❌ | ✅ 自动生成 |
| 广告 | 常见 | ❌ 无广告 |

## 总结

MaiImg 让图片分享变得可控。上传 → 设权限 → 分享链接或二维码，三步搞定。
