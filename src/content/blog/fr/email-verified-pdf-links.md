# Liens PDF vérifiés par e-mail : limiter l’accès via transfert

<div class="intro-panel">
  <p>Quand un lien PDF est transféré, vous perdez le contrôle. La vérification e-mail lie l’accès à une personne, pas à une URL.</p>
</div>

![Vérification e-mail pour l’accès](/diagram/en/email-verification-access-control-workflow.svg)

## Comment ça marche (simplement)

- Partager un lien
- Le lecteur saisit un e-mail
- Un code est envoyé
- Seuls les lecteurs vérifiés ouvrent le PDF

## Quand l’utiliser

- Propositions et devis
- Documents RH
- Livrables client
