# Upload PDF Get Link: Simple Sharing in Three Steps

Link sharing is easiest when the workflow stays minimal.

![Upload Interface](/2025MayMaiPDF/upload_in_cloudshare.png)

## Three steps

1. Upload PDF.
2. Get share link.
3. Share to recipients.

## Add safety defaults

- Set expiry for temporary docs
- Set open limit by audience size
- Disable download for sensitive files

## Final takeaway

Upload-link-share is the base workflow. Add small control settings and you get both speed and safety.
