2. Wait for processing
3. Gallery created
4. Ready for configuration

### Step 2: Configure View Limits

**Settings:**
1. Locate "View Limits" section
2. Enable "Limit total views"
3. Enter maximum view count
4. Review settings
5. Save configuration

**Recommendations:**

- Private sharing: 5-10 views
- Exclusive content: 1-3 views
- Business use: Custom limits

### Step 3: Set Time Controls

**Settings:**

3. Enter duration in seconds
4. Configure session behavior
5. Save settings

- Standard viewing: 60 seconds
- Extended access: 120+ seconds
- Custom durations available
