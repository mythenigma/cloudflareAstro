## 🚀 Cómo Funciona

### Proceso Simple

1. **Subir PDF**: Suba su documento PDF
2. **Generar Enlace**: Haga clic en "Generar Enlace"
3. **Obtener Código QR**: Obtenga código QR automáticamente
4. **Compartir**: Comparta enlace o código QR

### Opciones

- **Enlace Directo**: Comparta el enlace directamente
- **Código QR**: Use código QR para compartir
- **Ambos**: Use enlace y código QR
